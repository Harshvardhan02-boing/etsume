# Security Policy

## Supported Versions

Currently, only the latest release of Etsume is supported with security updates. We do not provide security updates for older versions.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Etsume, please do not open a public issue.
Instead, please contact the maintainers privately via email or through a GitHub Security Advisory if available.

We will try our best to address the vulnerability as soon as possible and release a patched version.
